# Todo List Component Assignment

Build a functional Todo List component in React that allows users to manage tasks effectively.

## Core Requirements

1. Create a Todo component with the following features:
   - Add new tasks
   - Edit existing tasks
   - Delete tasks
   - Show a message when no tasks exist

2. Implement state management using:
   - Tasks array
   - Input field value
   - Editing state

3. Required functionality:
   - Form submission handling
   - Input validation (no empty tasks)
   - Task editing mode
   - Task deletion

## Component Structure

```jsx
const Todo = () => {
  // State management
  // Task handling functions
  // JSX structure with:
    - Title
    - Form with input and button
    - Task list or "no tasks" message
}
```
